# 🚀 Process Scheduler Project

This project is a simulation of an Operating System Process Scheduler, implementing various scheduling algorithms.

## 📁 Project Structure
* **Backend:** Contains the server-side logic and API implementation.
* **Frontend:** The user interface for interacting with the scheduler.
* **Scripts:** Core scheduling algorithm scripts (e.g., FCFS, SJF, Banker's Algorithm, Round Robin).
* **Execution:** Use `start.bat` to launch the application quickly on Windows.
